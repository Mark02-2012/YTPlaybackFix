# YTPlaybackFix
<p>Fix playback issues by refreshing the video every time the error appears</p>
<p>Latest release = 0.2 (working), please test and report any issue https://github.com/Mark02-2012/YTPlaybackFix/releases</p>
<p><strong>IMPORTANT please if you want to integrate this tweak into your mod make sure you put the step to clone the branch "main" to build the stable version</strong></p>
